# Prerequisites for 158-bitcoin-oss

## Before You Start

You should have completed the entire course (01–134) and ideally built at least one of the Go capstone projects (130–134).

### What You Need

- Solid Go: functions, structs, interfaces, goroutines, error handling, testing
- Confidence reading unfamiliar Go code
- Git: branching, committing, pushing, pull requests
- A GitHub account

### Check Your Go Readiness

You are ready when you can:
- [ ] Open an unfamiliar Go file and explain what its functions do
- [ ] Write a table-driven test using `t.Run`
- [ ] Use `fmt.Errorf("context: %w", err)` for error wrapping
- [ ] Run `go test ./...` and interpret the output

If any of these feel uncertain, go back to `157-capstone` and build something non-trivial first.
