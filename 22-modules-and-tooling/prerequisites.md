# Prerequisites for 37-modules-and-tooling

## Before You Start

- You should have been using `go run .` in every challenge so far.
- [06-functions](../06-functions/skills.md) — basic Go code structure

## You're Ready When You Can...

- [ ] Create a new module with `go mod init`
- [ ] Add a dependency with `go get` and clean up with `go mod tidy`
- [ ] Run `go fmt ./...` and `go vet ./...` without errors
- [ ] Read a function's documentation with `go doc`
