## You're Ready When You Can...

- [ ] Declare a struct type with at least two fields
- [ ] Create a struct value using field names: `Person{Name: "Alice", Age: 30}`
- [ ] Write a method with a value receiver: `func (p Person) Method() string`
- [ ] Call a method: `p.Greet()`
