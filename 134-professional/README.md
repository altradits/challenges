## Open Source Contributions

### Getting Started

1. **Find beginner-friendly issues**
   - `good-first-issue` label
   - `beginner-friendly` label
   - Documentation improvements

2. **Popular FinTech-related projects**
   - `plutus` - Haskell-based FinTech
   - `fireblocks` - Crypto SDKs
   - `openbanking` - Open banking APIs
   - `financedatabase` - Financial data

3. **Your own projects**
   - Create tools for others
   - Document well
   - Accept contributions

### Contribution Workflow

```bash
# Fork and clone
git clone https://github.com/username/project.git
cd project

# Create branch
git checkout -b fix-bug-123

# Make changes
# ...

# Commit and push
git commit -m "Fix bug in calculation logic"
git push origin fix-bug-123

# Create pull request
```

## Job Application Strategy

### Resume Tips

- **One page** for <5 years experience
- **Quantify achievements** - "Built API serving 1000+ users"
- **Include projects** - Link to GitHub
- **Skills section** - List Go, JavaScript, Python prominently

### Cover Letter Template

```markdown
Dear Hiring Manager,

I'm a full-stack developer with expertise in building financial applications. I've built a complete FinTech portfolio including:

- Budget Planner API with Go and PostgreSQL
- Interactive frontend with React and TypeScript
- Data analysis service with Python and ML

I'm passionate about creating tools that help people manage their finances better. My experience with Go, JavaScript, and Python makes me a versatile candidate for your team.

I'd love to discuss how I can contribute to [Company Name]'s mission.

Best regards,
[Your Name]
```
